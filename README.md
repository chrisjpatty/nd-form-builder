# North Dakota Form Builder
A simple drag n' drop form builder to generate JSON objects to be used by the North Dakota Form Wizard.

### Browser Support
| Browser       | Minimum Version |
| ------------- |:---------------:|
| Chrome        | 4+              |
| Safari        | 4+              |
| Firefox       | 3.6+            |
| IE            | 11+ (Partial 9+ support)|
